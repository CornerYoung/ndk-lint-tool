# Nuo FE Lint Tool

[![NPM version][npm-image]][npm-url]
[![Downloads][downloads-image]][downloads-url]

Lint tool for Nuo FE.

## Install

```bash
npm i @nuofe/ndk-lint-tool -g
```

## Usage

```bash
ndk-lint
```
